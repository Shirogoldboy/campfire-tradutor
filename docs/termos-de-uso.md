## 2. Descrição do Serviço

O Campfire Tradutor é um aplicativo de tradução de arquivos, disponível para Windows (desktop)
e Android (mobile), que traduz textos, documentos, planilhas, legendas, imagens, áudio, vídeo
e outros formatos de arquivo usando uma combinação de:

- Um dicionário colaborativo público e gratuito;
- Serviços de tradução automática gratuitos de terceiros (MyMemory, LibreTranslate);
- O modelo de linguagem Claude, da Anthropic, acionado com a chave de API do próprio usuário
  quando necessário.

## 3. Chave de API e Custos de Terceiros

Determinados formatos e funcionalidades exigem que o usuário forneça sua própria chave de API
da Anthropic (e, opcionalmente, um token da Hugging Face para transcrição de áudio). O
Desenvolvedor **não fornece, vende ou intermedia** essas chaves e **não tem acesso** a elas além
do armazenamento local no dispositivo do usuário.

Qualquer custo de uso da API da Anthropic (ou de outros serviços de terceiros que exijam chave
própria) é de responsabilidade exclusiva do usuário, conforme os termos de preço do respectivo
provedor.
